# Hospital Performance Dashboard (Excel)

## Project Overview

This project presents an interactive Hospital Performance Dashboard built entirely in Microsoft Excel.

The dashboard helps analyze hospital operations, patient trends, revenue generation, appointment performance, and patient satisfaction using Excel's analytical features.

---

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPIs
- Conditional Formatting
- Data Cleaning
- Excel Formulas

---

## KPIs

- Total Revenue
- Total Patients
- Average Patient Satisfaction
- Frequent Missers
- Patients with No Missed History
- No Show Rate

---

## Dashboard Features

- Department-wise Revenue
- Department-wise Patient Count
- Appointment Status Analysis
- Revenue Analysis
- Patient Satisfaction
- Interactive Slicers
- Dynamic KPIs

---

## Dataset

The dataset contains hospital records including:

- Patient ID
- Department
- Appointment Status
- Total Bill
- Patient Satisfaction
- Insurance
- Age Group
- Waiting Time
- City
- Chronic Disease
- Appointment Type
